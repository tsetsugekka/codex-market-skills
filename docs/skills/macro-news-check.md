# Macro News Check

`macro-news-check` 是给其他市场 skill 调用的宏观快讯检查 skill。它只在分析确实需要当前宏观、大盘或跨资产背景时使用，重点查看金十、华尔街见闻和 FinancialJuice 的公开快讯/页面/Feed；遇到 A 股大盘盘面、板块轮动、买什么方向或是否入场问题时，先看快讯，再用搜狐指数、行业/概念板块涨跌幅和涨跌停历史做辅助确认；遇到日本市场强弱、日经/TOPIX 拖累来源或日本个股受大盘压制时，先看快讯，再用 JPX 实时指数/行业数据做辅助确认，并把相关消息转化为对股票、指数、技术结构或 gamma 结构的影响判断。

## 能力

- 检查利率、汇率、央行、经济数据、商品、地缘、指数期货和 risk-on/risk-off 快讯。
- 优先使用金十作为中文宏观 tape；华尔街见闻作为中文/亚洲市场补充；FinancialJuice 作为英文全球快讯补充。
- A 股盘面问题保持“快讯优先、搜狐辅助”：快讯解释政策/宏观/外围/突发风险，搜狐板块涨跌幅验证资金落点、扩散程度和强弱分化。
- 日本盘面问题保持“快讯优先、JPX 辅助”：快讯解释日债、汇率、BOJ/MOF、外围科技和风险情绪，JPX 实时指数/TOPIX 33 業種/TOPIX-17 验证拖累或支撑来自哪些行业与风格。
- 判断宏观消息是主因、二级放大器，还是背景噪音。
- 将宏观 tape 和具体标的连接起来，例如日本股对应日债收益率、USD/JPY、日经/TOPIX，A 股对应政策流动性、人民币和大盘情绪，美股 gamma 对应 FOMC/CPI/美债/美元/油金。
- 只保留和当前分析相关的 2-5 条重点消息，避免堆砌快讯列表。


```text

```

```text
这个 SPX gamma 图要不要考虑今天的宏观快讯？
```

```text
日本股今天是不是被日债收益率和日经拖累？
```

## 公开安全说明

这个 skill 只使用公开页面、公开 Feed 或公开接口，不读取登录 cookie、token、账号数据或私有研究资料。输出时只做标题级摘要和市场影响判断，不复制长篇新闻正文，不保存原始快讯流，不把个人关注列表、`.env`、截图、缓存或私有输出提交到公开仓库。
