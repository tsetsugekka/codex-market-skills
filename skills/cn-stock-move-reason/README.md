# CN Stock Move Reason

`cn-stock-move-reason` 是一个用于 Codex 的 A 股异动解析 skill。它针对用户输入的单只 A 股代码，从东方财富公开行情、公告、股吧/资讯帖，以及搜狐指数/板块页面、A 股涨跌家数背景中收集证据，再由 Codex 分析个股异动原因和短线情绪周期位置。

这个目录里的 `SKILL.md` 是给 Codex 执行时读取的规则文件；本 `README.md` 是给 GitHub 用户阅读的公开说明。

## 能做什么

- 获取 A 股当前价格、涨跌幅、振幅、换手率、成交额、市值、PE/PB 等公开行情数据。
- 获取东方财富公告列表，优先识别业绩、回购、增减持、重组、订单、中标等确认材料。
- 默认抓取最多 100 条东方财富股吧/资讯帖，用于定性观察市场讨论焦点和情绪面。
- 获取搜狐证券指数、行业板块、概念板块背景；今日盘中涨跌家数优先参考东方财富指数页 quote 接口（`push2delay.eastmoney.com`），历史涨跌停/涨跌家数参考搜狐 `zdt.shtml`，用于判断大盘、板块、个股是否同步共振。
- 按冰点、修复/潜伏、启动、加速、高潮、高位分歧/分化、退潮七段分析短线情绪位置，并区分健康分歧和高位分化。




## 脚本用法

在仓库根目录运行：

```bash
python3 skills/cn-stock-move-reason/scripts/stock_move_sources.py 600519 --format markdown
```

常用参数：

- `--hours 24`：材料时间窗口。
- `--posts 100`：最多收集的东方财富股吧/资讯帖数量，默认 100。
- `--announcements 10`：最多收集的公告数量。
- `--skip-market-context`：跳过大盘指数、板块和涨跌家数背景，只输出个股材料。

## 需要的权限和来源

- 只读取公开网页或公开 API。
- 指数、行业板块、概念板块优先使用搜狐证券公开页面；今日涨跌家数使用东方财富指数页公开 quote 接口（`push2delay.eastmoney.com`），历史涨跌停/涨跌家数使用搜狐证券 `zdt.shtml`。
- 不读取本地 token、Cookie、`.env` 或其他凭据。
- 不写入任何外部服务。
- 不调用 Gemini、OpenAI API 或其他 LLM API；分析由当前 Codex 会话完成。

## 公开安全说明

这个 skill 和脚本可以放在公开仓库中。请不要提交个人关注列表、私有分析输出、运行缓存、`.env`、token 或任何含有登录信息的文件。

本 skill 输出仅用于个人研究，不构成投资建议。股吧内容只代表公开社区发言，不能当作事实依据。
